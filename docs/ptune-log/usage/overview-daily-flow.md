## 1日の流れ（全体像）

Obsidian とスマホアプリ ptune を連携して、  
1日の作業を **5つのステップ**で進める流れを紹介します。

![1日の活動フロー](./images/daily_flow.png)

---

## 5つのステップ概要

- **① 朝：タスク定義**  
   デイリーノートにタスクを作成し、Google Tasks／ptune と同期します。

- **② 昼：作業開始**  
   タイマーを開始し、その日の作業ノートを準備します。

- **③ 作業記録：ノート記録**  
   様々な作業内容の記録をノートに整理しながら進めます。

- **④ 作業終了**  
   LLM レビューで内容をまとめ、次の作業への準備をします。

- **⑤ 夜：振り返り**  
   タスク実績を取り込み、1日の振り返りレポートを作成します。

---

## 注意事項

- **③ と ⑤ はプロトタイプ機能のため、今後仕様が変わる可能性があります。**
- 全体の流れは変更されません。
