## システム要件

### 対応OS
- Windows 11
- macOS 13 以降

### Google アカウント
- Google Tasks と連携してタスク管理を行います
- Google アカウントの利用は無料です
- スマホタイマーアプリ **ptune（無料）** が必要です  
  （Google Tasks と同期するため）

### AI サービス API キー
- ノートの要約やキーワード抽出に利用します
- 以下のいずれかの API キーが必要です
  - OpenAI API キー
  - Google Gemini API キー
  - Anthropic Claude API キー
- 各サービスは **従量課金制の有償サービス** です
