## セットアップ概要

この章では、Obsidian で **ptune-log プラグイン** を利用するための  
セットアップ手順を説明します。

Windows と macOS では必要な作業が異なります：

- **Windows：タスク同期アプリ「PtuneSync」をインストール**
- **macOS：Google OAuth クライアント設定を行う**

---

### PtuneSync とは？（Windows ユーザー向け）

PtuneSync は、Obsidian で管理しているタスクと  
Google Tasks（スマホアプリや Web と共有されるタスク管理）を  
自動で同期するための **Windows 専用アプリ** です。

主な役割：

- Google 認証設定を自動化  
- Obsidian ↔ Google Tasks 間のタスクを安全に連携  
- スマホアプリ **ptune** の作業ログと連動

Windows では PtuneSync をインストールするだけで  
Google Tasks とのタスク同期がそのまま利用できます。
