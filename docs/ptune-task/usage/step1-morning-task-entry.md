## ① 朝: デイリーノートにタスクを書く

朝にやる最初の作業は、**今日の予定タスクを整理してデイリーノートに書くこと** です。

## 1. 今日のデイリーノートを開く

1. Obsidian を開く
2. 今日のデイリーノートを開く
3. `_journal/yyyy-mm-dd.md` が開くことを確認する

このデイリーノートは、朝の計画と夜の振り返りの両方で使います。

## 2. 今日の予定タスクを書く

`今日の予定タスク` の見出しに移動して、今日やることを書きます。

書き方の例:

- 親タスクを先に書く
- 必要に応じてサブタスクに分ける
- `🍅x1` のように予定ポモドーロ数を付ける
- `🚫` は記録を残さなくてよい雑タスクに使う
- `<朝>` のような時間帯ラベルは任意で使う

## 3. チュートリアル用の例

```markdown
## ✅ 今日の予定タスク

- [ ] <朝>くすり🚫
- [ ] ptuneストア公開更新
  - [ ] gitマージ #実装 🍅x1
  - [ ] Android版 #実装 🍅x1
  - [ ] iPhone版 #実装 🍅x1
```

## 補足: コード補完を使って入力する

- タスク入力中に `::` を入れると、予定ポモドーロ数の候補を補完できます
- `##` を入れると、タグ候補を補完できます
- `||` を入れると、完了目標の候補を補完できます
- `@@` を入れると、サブタスクのテンプレート候補を補完できます
- 候補の内容は `日次タスク設定` のテンプレートや設定内容に応じて変わります
- まず候補を使って入力し、必要な部分だけ手で調整する進め方を推奨します

## 4. Google Tasks に同期する

ptune と ptune-task は Google Tasks を介して連携します。

1. Ctrl+P で、Obsidian のコマンドパレットを開く
2. `ptune-task : 今日の予定を Push` を実行する
3. スマホアプリ ptune を開く
4. 同期画面で `タスクを取り込む` を実行する

これで朝の準備は完了です。

## 補足: 前日の未完了タスクの事前取込み

- 事前に Google Tasks にあるタスクを 予定タスクに取込みが可能です  
- Ctrl+P で、`ptune-task : 今日の予定を Pull` を実行します  
- 未完了タスクのみ Google Tasks 側からタスクの取込みをします  
- Google Tasks に直接登録したタスクも、 Pull で取込みをしてください
