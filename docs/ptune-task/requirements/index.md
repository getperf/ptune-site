## システム要件

### 対応 OS

- Windows 11

### 必要なアプリとアカウント

- Obsidian デスクトップ版
- Google Tasks と連携するための Google アカウント
- スマホ側で利用する `ptune` アプリ

※ スマホアプリ `ptune` は公開審査中のため、[ユーザテスト](/user-test/) のページから試使用版をインストールしてください

### 補助ツール

- Google 認証と同期には **PtuneSync** を利用します
- PtuneSync は Windows 専用です

### AI サービス API キー

- ノートの要約やキーワード整理を使う場合は API キーが必要です
- 対応サービス
  - OpenAI API キー
  - Google Gemini API キー
  - Anthropic Claude API キー
- AI 機能は任意です
