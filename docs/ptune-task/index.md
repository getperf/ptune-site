## Obsidian プラグイン ptune-task ガイド

本ガイドは、Obsidian プラグイン **ptune-task** のセットアップ手順をまとめたものです。

ptune-task は、`ptune` と Google Tasks を組み合わせて、
1日のタスク管理と実績の振り返りを行うための基本機能です。

このフェーズでは、まず次の内容を案内します。

- 動作前提
- Obsidian の準備
- BRAT を使った ptune-task の導入
- PtuneSync を使った Windows 環境の認証と同期設定

利用手順やチュートリアルは、別イテレーションで見直します。
