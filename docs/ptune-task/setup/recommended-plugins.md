## 推奨項目の各種プラグイン設定

このページでは、セットアップ確認モーダルの **推奨項目** に表示されるプラグインを説明します。

これらは必須ではありませんが、運用しやすくするために導入を推奨します。

### 1. 設定手順

1. `Ctrl + P` でコマンドパレットを開き、 **セットアップ確認** モーダルを開きます。
2. プラグインタイトルの右側のリンクを開いてインストールして有効化する
3. モーダルを開き直して状態が更新されることを確認する

### 補足：推奨プラグインについて

#### Calendar

- デイリーノートを日付から開きやすくする
- 日次運用の移動を簡単にする

#### Outliner

- 箇条書きやタスクのインデント編集をしやすくする
- Obsidian 上で日々のタスク整理を行いやすくする
- **今日の振り返り** **「振り返りポイント」レポート** の編集で利用します

#### Bases

- SQL ライクな言語で、 Obsidian のノートの検索用ビューを表示する
