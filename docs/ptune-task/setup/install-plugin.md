## ptune-task プラグインインストール

この章では、Obsidian に **ptune-task** を導入する手順を説明します。
フェーズ1では、**BRAT を使った導入** を標準手順として扱います。

## 1. BRAT をインストールする

BRAT は、コミュニティプラグインとして未公開またはベータ版のプラグインを
GitHub リポジトリから導入するための Obsidian プラグインです。

1. Obsidian の **設定 > コミュニティプラグイン** を開く
2. **閲覧** をクリック
3. 検索バーに `BRAT` と入力
4. **Obsidian42 - BRAT** をインストール
5. インストール後に **有効化** する
6. 有効化を確認したら、設定画面を閉じます

## 2. BRAT 経由で ptune-task を追加する

1. `Ctrl + P` でコマンドパレットを開く
2. 検索バーに `Add a Beta plugin` を入力し、表示されたコマンドを実行
3. GitHub リポジトリとして `getperf/ptune-task` を入力
4. **Select a version** に `Latest` を選択
5. **Add plugin** をクリックして追加を実行
6. インストール完了後、 **設定 > コミュニティプラグイン** を開く
7. コミュニティプラグイン一覧に `ptune-task` が表示されることを確認
8. 確認したら、設定画面を閉じます

## 3. 初期セットアップを実行する

ptune-task では、初回利用時にノート環境の準備や関連設定をまとめて進める前提です。

1. `Ctrl + P` でコマンドパレットを開く
2. `ptune` で検索し、「セットアップ確認」のコマンドを選択
3. 画面の案内に従ってノート保存先や必要設定を行います

## 4. セットアップ時の確認ポイント

- `ptune-task` がコミュニティプラグイン一覧で有効になっている
- Vault 配下に必要なノート用ディレクトリやテンプレートが作成されている
- デイリーノートや関連プラグインの設定が反映されている
- 次の手順として PtuneSync のセットアップに進める

## 注意

- フェーズ1では ZIP 配布や手動配置は案内しません
- プラグイン更新も BRAT 経由の更新を前提にします
- 画面名やボタン名は Obsidian / BRAT のバージョンで多少異なることがあります
