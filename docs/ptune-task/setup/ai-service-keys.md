## AI サービスキー設定

このページでは、ptune-task の AI 補助機能を利用する場合に必要な API キー設定を説明します。

AI 機能は任意です。まずは Obsidian、ptune-task、PtuneSync の基本セットアップ完了を優先してください。

### 対応サービス

- OpenAI
- Google Gemini
- Anthropic Claude

### 基本手順

1. 利用するサービスの API キーを取得する
2. Obsidian の **設定 > ptune-task** を開く
3. AI 関連の設定項目に API キーを入力する
4. 必要なら利用モデルやプロンプト設定を調整する

### 注意

- API キーの発行や利用料金は各サービスの契約条件に従います
- AI 機能を使わなくても ptune-task の基本運用は可能です
