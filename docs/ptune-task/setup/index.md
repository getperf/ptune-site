## セットアップ概要

この章では、Obsidian で **ptune-task プラグイン** を使い始めるまでの準備を説明します。

フェーズ1では、Windows 環境を前提に手順を整理しています。

### セットアップの流れ

1. Obsidian をインストールし、Vault を作成する
2. コミュニティプラグインを有効化する
3. BRAT をインストールする
4. BRAT 経由で `getperf/ptune-task` を追加する
5. ptune-task の初期設定を行う
6. PtuneSync をインストールして Google 認証を完了する
7. 必要に応じて AI サービスや補助設定を追加する

### BRAT とは

BRAT は、Obsidian の非公式プラグインをインストール・管理するためのコミュニティプラグインです。ptune-task などの公式ストアに登録されていないプラグインを簡単に追加できます。2026年4月現在、公式ストア登録申請中で、承認されるまでは BRAT を使用して ptune-task をインストールします。

### PtuneSync とは

PtuneSync は、Obsidian と Google Tasks の認証と同期を補助する Windows アプリです。

- Google 認証フローの実行
- ptune-task と Google Tasks の接続補助
- `ptune` アプリと Obsidian の連携前提を整備
