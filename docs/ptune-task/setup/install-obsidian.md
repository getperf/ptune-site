## Obsidian インストール

ここでは、Windows 11 に Obsidian をインストールし、ptune-task 用の Vault を準備する手順を説明します。

## 1. Obsidian をインストールする

1. [公式サイト](https://obsidian.md/download) にアクセス
2. Windows Installer をダウンロード
3. ダウンロードした `Obsidian-Setup.exe` を実行
4. 画面の指示に従ってインストール
5. Obsidian を起動

## 2. Vault を作成する

Obsidian では、ノート保存フォルダを **Vault** と呼びます。
ptune-task のノートや設定は、この Vault 配下に保存されます。

1. 起動直後の画面で **新しい Vault を作成** を選択
2. Vault 名を入力
3. 保存先フォルダを指定
4. 作成後、Vault が開くことを確認

Vault の実体は通常のフォルダです。
Markdown ファイルや `.obsidian` 配下の設定もエクスプローラーから確認できます。

## 3. コミュニティプラグインを有効化する

ptune-task と BRAT を導入するには、コミュニティプラグインを有効化する必要があります。

1. 左下の **設定** を開く
2. **General > Language** で日本語を選択
3. **コミュニティプラグイン** を開く
4. **コミュニティプラグイン** を有効化にする
6. **閲覧** ボタンが表示されることを確認

### 注意

- コミュニティプラグインは自己責任で利用します
- Obsidian のバージョン差により表示名が少し異なる場合があります
