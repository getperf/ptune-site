## ptune-task 設定の調整

ptune-task 設定で、初期セットアップで調整必要な次の 2 つを設定します。

- レビュー設定
- 日次タスク設定

## 1. 設定画面を開く

1. Obsidian を開く
2. 左下の **設定** を開く
3. サイドバーのコミュニティプラグイン一覧から **ptune-task** を選択する

## 2. レビュー設定の調整項目

レビュー設定では、当日作成ノートレビューや振り返りポイントの出力形式を調整します。

### 当日作成ノートレビュー

- 無効にすると当日作成ノートの分析を省略し、タスク実績の分析のみ実施します
- まずは有効のまま使い始めて問題ありません
- ノートの記録をしない場合は無効にしてください

### 振り返りポイント形式

- 振り返りポイントの既定出力形式を選びます
- 通常は `outline` (アウトライン形式のテキスト編集)を選択します
- **XMind を利用しているユーザは `xmind` を推奨**します
- **XMind** 上で振り返りを実行します
- XMind はマインドマップ形式で考えを整理したい場合に向いています
- 参考: [XMind 8 ダウンロード](https://xmind.komodo.jp/download/xmind8)

## 3. 日次タスク設定の調整項目

日次タスク設定では、朝タスク、夜タスク、タグ候補、ゴール候補、サブタスクテンプレートなどの初期候補をまとめて設定できます。

### テンプレートダイアログを開く

1. **日次タスク設定** の **テンプレートを開く** をクリックする
2. **日次タスクテンプレート** ダイアログが開くことを確認する
3. 利用したい **タスクテンプレート** を選択する
4. 必要に応じて **Habit タスクも更新する** を切り替える
5. **適用** をクリックする

これにより、日次タスク設定の各入力欄にテンプレート内容が反映されます。

### 補足

- まず既定のテンプレートを適用してから、必要な箇所だけ個別に微調整する進め方を推奨します
- 入力欄は改行区切りで候補を管理します
