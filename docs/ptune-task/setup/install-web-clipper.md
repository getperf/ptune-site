6. **Web Clipper の確認**

Obsidian Web Clipper は、Web ページを Obsidian のノートとして保存できる公式拡張機能です。  
まず以下のページからインストールします。

- **Web Clipper インストールページ**  
  https://obsidian.md/clipper

---

### インストール後の動作確認（利用例）

1. Web Clipper をインストールしたブラウザで **ptune サイト**を開きます。  
   https://ptune.getperf.net/

2. ブラウザの拡張機能メニューから  
   **Obsidian Web Clipper** を起動します。

3. 表示される画面で **「Obsidian に追加」** をクリックすると、  
   Web ページが Obsidian の Vault 内 **`Clippings/` フォルダ**に保存されます。

4. Obsidian を開き、  
   `Clippings/ホーム - ptune.md`（例）  
   が作成されていることを確認します。
