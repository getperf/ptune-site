## 設定一覧

ptune-task の主な設定項目を、用途別にまとめます。

## 一覧

| 設定グループ | 主な項目 | 定義 |
| --- | --- | --- |
| 基本設定 | 言語、ログレベル、ログファイル出力 | 表示言語やログ出力の基本動作 |
| LLM 設定 | プロバイダー、API キー、Base URL、モデル、Temperature、Max Tokens | ノート要約などで使う LLM の接続先や生成条件 |
| ノート設定 | フォルダ接頭辞、ノート接頭辞、連番の桁数、テンプレートファイル | プロジェクトフォルダやノートの作成ルール |
| プロジェクトインデックス設定 | Bases セクションを有効にする | プロジェクト index の表示補助 |
| レビュー設定 | 当日作成ノートレビュー、振り返りポイント形式、レビュー傾向の日数、XMind テンプレートパス | 夜の振り返りレポートの既定動作 |
| 日次タスク設定 | テンプレート、朝のタスク、夜のタスク、タグ候補、ゴール候補、サブタスクテンプレート | 毎日のタスク入力や補完候補の内容 |

## 基本設定

### `言語`

- 設定画面の表示言語を切り替えます
- `日本語` と `英語` を選べます

### `ログレベル`

- ログの出力レベルを切り替えます
- `debug` `info` `warn` `error` `none` を選べます

### `ログファイル出力`

- ログをファイルへ保存するかを切り替えます
- ログは Obsidian Vault 下の `.obsidian/plugins/ptune-task/logs` に保存します

## LLM 設定

### `プロバイダー`

- ノート要約などで使う LLM サービスを選択します
- `OpenAI` `Claude` `Gemini` `Custom` を選べます

### `API キー`

- 利用する LLM サービスの API キーを設定します

### `Base URL`

- API の接続先 URL を設定します
- `Custom` 構成や互換エンドポイントを使うときに調整します

### `モデル`

- 要約生成に使うモデル名を設定します
- 応答速度、品質、コストに応じて使い分けます

### `Temperature`

- 要約生成のゆらぎを調整します
- 出力を安定させたいときは低め、表現の幅を持たせたいときは高めにします

### `Max Tokens`

- 要約生成で使う最大トークン数を設定します
- 長めのノートを扱うときや、出力量を抑えたいときに調整します

## ノート設定

### `フォルダ接頭辞`

- プロジェクトフォルダ名の付け方を設定します
- `連番` または `日付` を選べます

### `ノート接頭辞`

- プロジェクト配下のノート名の付け方を設定します
- `連番` または `日付` を選べます

### `連番の桁数`

- 連番を使うときのゼロ埋め桁数を設定します

### `テンプレートファイル`

- 新規ノート作成時に使うテンプレート内容を設定します

## プロジェクトインデックス設定

### `Bases セクションを有効にする`

- Bases プラグインを使う場合に、プロジェクトフォルダの index.md へ Bases 用セクションを追加または補完します
- Bases 用セクション はプロジェクト下のノート一覧を表示します

## レビュー設定

### `当日作成ノートレビュー`

- `今日の振り返り Review` 実行時に、当日作成ノートレビューを既定で有効にするかを設定します
- ノートの記録をしない利用の場合は無効にします

### `振り返りポイント形式`

- 振り返りポイントの既定出力形式を設定します
- `outline` または `xmind` を選べます

### `レビュー傾向の日数`

- 日次傾向レポートに含める日数を設定します

### `XMind テンプレートパス`

- XMind 形式で振り返るときに使うテンプレートファイルの vault 内パスを設定します
- 既定は空の XMind ファイルになります
- 独自テンプレートをカスタマイズしたいときに調整します

## 日次タスク設定

### `テンプレートを開く`

- `テンプレートを開く`　をクリックし、あらかじめ用意されたテンプレートを使って、毎日のタスク設定を初期化します

### `朝のタスク`

- 毎朝確認する生活系タスクを改行区切りで設定します

### `夜のタスク`

- 毎晩確認する生活系タスクを改行区切りで設定します

### `タグ候補`

- タスク入力時のタグ候補を設定します
- 予定タスクの入力で、`##` で呼び出す補完候補

### `ゴール候補`

- タスク完了目標の候補を設定します
- 予定タスクの入力で、`||` で呼び出す補完候補

### `サブタスクテンプレート`

- サブタスク入力時のテンプレート候補を設定します
- 予定タスクの入力で、`@@` で呼び出す補完候補

## 日次タスク設定の使い方

- Obsidian の `設定` から `ptune-task` の設定画面を開きます
- `日次タスク設定` の `テンプレート` で `テンプレートを開く` をクリックします
- 使用したいテンプレートを選択します
- 必要に応じて `Habit タスクも更新する` を選びます
- `適用` をクリックして日次タスク設定へ反映します
- 反映後に `朝のタスク` `夜のタスク` `タグ候補` `ゴール候補` `サブタスクテンプレート` を必要に応じて調整します
