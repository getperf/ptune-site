# ユーザテストへのご協力

本ページでは、ユーザテストのセットアップ方法・インストール導線・実施内容をまとめています。  

正式公開前の最終チェックのため、ぜひご協力をお願いします。
本ソフトウェア（**ptune / ptune-log / PtuneSync**）は、すべて **フリーソフトとして無料で自由に試せます**。  

今年度、**リリース説明会（オンライン）を開催予定**で、その場でもフィードバックをいただけます。  

参加希望の方は furu3@getperf.net にお問い合わせください。

---

## 1. テスト対象パッケージの概要

| パッケージ | 役割 | テスト状況 | インストール方法 |
|------------|------|-------------|------------------|
| **ptune（スマホアプリ）** | タイマー・タスク管理 | Android/iPhone クローズドテスト中 | → [Android](#android) / [iPhone](#iphonetestflight) |
| **ptune-log（Obsidian プラグイン）** | 作業ログ記録・整理・振り返り | Obsidian 公式レビュー待ち | → [Obsidian（ptune-log）](#obsidianptune-log) |
| **PtuneSync（Windows）** | Obsidian と Google Tasks の連携補助 | 公開済み | → [Windows（PtuneSync）](#windowsptunesync) |

---

## 2. インストール方法（環境別）

### Android（クローズドテスト参加）
- **Gmail アドレスの事前登録が必要です**  
  → 製品サポート用アドレス（furu3@getperf.net）へメールで登録依頼してください  
- 登録後、以下からアプリをインストール  
  - [Play ストア（ptune）](https://play.google.com/store/apps/details?id=net.getperf.ptune)

---

### iPhone（TestFlight）
- 事前登録不要  
- TestFlight アプリをインストール  
- 次に以下リンクを開いて参加  
  - [TestFlight（ptune）](https://testflight.apple.com/join/q81BYpyS)

---

### Windows（PtuneSync）
- Microsoft Store からインストール  
  - [PtuneSync（Microsoft Store）](https://apps.microsoft.com/detail/9p9hqs78nvlz?hl=ja-JP&gl=JP)

---

### Obsidian（ptune-log）
- 以下の **公式ガイドのセットアップページ** に従って、ZIP を手動インストール  
  - [ptune-log セットアップガイド](https://ptune.getperf.net/ptune-log/setup/)  
※ ガイド内の「Obsidian インストール」～「その他のセットアップ」までの作業が必要です。

---

## 3. テストで協力していただきたい内容

1. 本ページに沿って、各アプリをセットアップ  
2. **利用手順ガイド（朝 → 昼 → 夜）** を参考に、1日の作業フローを実施  
3. スムーズに利用できるか、迷う操作がないかを確認  
4. 不具合・改善点・ご意見をフィードバック  

[利用手順概要はこちら](https://ptune.getperf.net/ptune-log/usage/)
