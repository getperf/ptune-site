    プライバシーポリシー

    アプリ名: ptune
    開発者: 古澤 実
    サポート連絡先: furu3@getperf.net
    ドメイン: https://ptune.getperf.net
    公開日: 2025年9月13日
    最終改定日: 2025年11月19日

    1. 対象アプリ

    本ポリシーは以下のソフトウェアを対象とします。

    • モバイルアプリ版 ptune
    • デスクトップ版 PtuneSync
    • Obsidian プラグイン（ptune-log）

    これらはいずれもユーザーの端末上で動作し、開発者の独自サーバーは利用しません。

    2. 取得する情報

    2-1. Google OAuth を通じて取得する情報（モバイル版 / デスクトップ版）

    本アプリは Google OAuth 認証により、以下の最小限の情報を取得します。
    これらは Google Tasks の同期機能のためだけに利用されます。

    • Google Tasks のタスク情報
    • タイトル、ID、状態、期日、メモなど（Google Tasks API 標準項目）
    • Google アカウントの基本情報（メールアドレス）

    2-2. 端末内で生成されるデータ

    • ポモドーロタイマーの記録
    • タスクの完了履歴
    • 振り返りレポートのメタデータ
    • Obsidian ノート（ローカル保存ファイル）

    2-3. 外部 AI サービス利用時に送信される情報（任意設定）

    ノートのタグ生成・要約を行う場合、ユーザーが入力した API キーにより、
    ノート本文の一部が OpenAI 等の外部 AI API に直接送信されます。

    • 送信されるのはユーザーが作成したノート本文の必要部分のみ
    • 開発者がデータを受け取ることはありません
    • API キーは端末内にのみ保存され、外部送信されません

    3. 利用目的

    3-1. Google OAuth で取得するデータの利用目的

    • Google Tasks の表示・同期・管理
    • タスクとポモドーロタイマーの連携
    • 自動レポート生成（ユーザー端末内での処理）

    3-2. Obsidian ノートの利用目的

    • タスク記録や作業ログのローカル管理
    • ユーザーの振り返りデータのローカル保存

    3-3. 外部 AI サービスの利用目的（任意設定）

    • ノートのタグ生成
    • ノートの要約作成

    ※ユーザーが API キーを設定しない場合、AI 機能は動作しません。

    4. データの保存について

    • 本アプリは 独自サーバーにデータを保存しません
    • Google Tasks は Google のクラウド上に保存されます
    • 振り返り JSON / ノートデータはローカル端末のフォルダに保存されます
    • モバイルアプリのキャッシュはログアウトまたはアプリ削除時に削除されます
    • API キーは端末ローカルに保存され、開発者へ送信されません

    5. データ共有・第三者提供

    • 開発者（古澤 実）を含む第三者へデータを提供することはありません
    • 広告会社・分析事業者・外部サービスへの自動共有はありません
    • AI サービス利用時は、ユーザーとサービス提供元（OpenAI 等）が直接通信します
    • 開発者サーバーを中継しません

    6. ユーザーによる制御方法

    • Google OAuth の許可を取り消す場合は
      Google アカウントのアクセス許可ページ を利用できます
    • モバイルアプリの「ログアウト」操作で
      アクセストークンとキャッシュが削除されます
    • Obsidian プラグインでは、API キーを削除することで
      外部 AI 送信がただちに停止します
    • ノートやレポートはユーザーが自由に編集／削除できます

    7. セキュリティ対策

    • 本アプリは独自サーバーを介さず、Google API と HTTPS で直接通信します
    • OAuth トークンは端末内の安全な領域にのみ保存されます
    • Obsidian プラグインはローカルファイルにのみアクセスします
    • AI API との通信はユーザー自身の API キーにより直接行われます
    • いずれのデータも開発者に送信されることはありません

    8. ノートのタグ生成・要約機能について（外部AIサービス）

    • この機能はユーザーが任意で利用できるオプション機能です
    • ノート本文の一部を外部 AI サービスへ送信して解析を行います
    • 解析内容はタグ生成と要約に限定されます
    • Google Tasks のデータは AI へ送信されません
    • API キーはユーザーの端末内に保存され、外部共有はありません
    • 開発者は解析内容や送信データにアクセスできません

    9. 免責事項

    • 本アプリはオープンソースとして無償提供されています
    • 開発者は予告なく機能の変更や停止を行う場合があります
    • 本アプリの利用により発生した損害について、開発者は責任を負いかねます
